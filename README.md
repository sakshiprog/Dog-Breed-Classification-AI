# 🐶 Dog Breed Classification AI

A Flask-based web application that predicts dog breeds from uploaded images using a pre-trained MobileNetV2 deep learning model.

---

## 🚀 Features
- Upload dog images from browser
- Real-time breed prediction
- Confidence score output (%)
- Simple and interactive UI
- End-to-end AI + Web integration

---

## 🧠 Tech Stack
- Python
- Flask
- TensorFlow / Keras (MobileNetV2)
- NumPy
- Pandas
- HTML / CSS

---

## 📂 Project Structure
app.py  
model.py  
labels.csv  
static/  
templates/  
test/  

---

## ⚙️ How It Works
1. User uploads an image via web interface  
2. Image is preprocessed for MobileNetV2  
3. Pretrained model predicts the dog breed  
4. Output is displayed with confidence score  

---

## ▶️ Run Locally

### 1. Clone repository
git clone https://github.com/sakshiprog/Dog-Breed-Classification-AI.git  
cd Dog-Breed-Classification-AI  

### 2. Install dependencies
pip install -r requirements.txt  

### 3. Run application
python app.py  

### 4. Open in browser
http://127.0.0.1:5000
